# SaludIA Landing Page Este es el sitio web profesional de **SaludIA**, un chatbot inteligente para clínicas dentales. ## 📂 Archivos - `index.html`: Página principal - `styles.css`: Estilos ## 🚀 Cómo publicar gratis ### GitHub Pages 1. Crea un repositorio público en GitHub. 2. Sube `index.html` y `styles.css` a la rama `main`. 3. Ve a **Settings > Pages**. 4. Elige la rama `main` y la carpeta `/root`. 5. Guarda y GitHub generará la URL: `https://tuusuario.github.io/saludia-web` ### Netlify 1. Crea una cuenta en [Netlify](https://netlify.com). 2. Conecta tu repositorio de GitHub. 3. Haz Deploy automático. 4. Obtén un subdominio gratuito como `saludia.netlify.app`. --- **Contacto:** contacto.saludia@gmail.com © 2025 SaludIA
